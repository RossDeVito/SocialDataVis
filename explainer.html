Motivation





My dataset has two main parts. The neighbothood data is a combination of the shapefiles of the Neighborhood Tabulation Areas (https://data.cityofnewyork.us/City-Government/Neighborhood-Tabulation-Areas/cpf4-rkhq) and the tables from the Demographics and profiles at the Neighborhood Tabulation Area (NTA) level (https://data.cityofnewyork.us/City-Government/Demographics-and-profiles-at-the-Neighborhood-Tabu/hyuz-tij8). The school data is a combination of the data from the 2017 DOE High School Directory (https://data.cityofnewyork.us/Education/2017-DOE-High-School-Directory/s3k6-pzi2) and the 2016-17 School Quality Reports Results for high school (http://schools.nyc.gov/Accountability/tools/report/default.htm#Citywide).

I chose this dataset because this was something I discussed in high school in upstate New York, and I now go to school with many people who went through this system while all the changes were being made.

My goal for the end user's experience was to be able to see the general trends as far as what is making a difference in school performance, but also what schools or types of schools are able to break these trends. That's why schools are shown geographically, are shown by type, and can also be individually selected for more details.

Basic Stats

The dataset I am currently working with is greatly cut down from the original sources. In selecting what data to keep the aim was to find attributes which might be good incicators of school performance either as metrics of performance or challenges overcome.

I did exploratory data analysis using an earlier version of the dataset and seaborn, a python package. The results were that most plots had a trend with notable outliers, so that steered me towards making this visualization potentially a tool for identifying and exploring those outliers.

Genre

The overall genre of this visualization is partitioned poster.

This project uses zooming on the map to both provide overall feels for areas in addition to being able to see detailed areas. A constant visual platform and object consistency are also used. The map and scatter plot remain the same even when things arounf them change, and the schools are always circles across both parts with a square when selected.

The project is very much user directed, allowing the user to change the variable and then hover and click for details on demand. There is also filtering by type to allow for type comparisons.

Discussion

I'm happy with how this project very much resembles the original goal, but if I were to do it again I would provide some more numerical analysis. That would provide even more information to use to compare schools or groups of school to determine what is best,
